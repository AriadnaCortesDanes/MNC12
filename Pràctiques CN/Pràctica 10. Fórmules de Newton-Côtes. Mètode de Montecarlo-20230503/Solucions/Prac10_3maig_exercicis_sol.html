a = 0; b = 1;

h = 1./2.^k;

Pràctica 25. Mètodes de Montecarlo

  • Calculeu per fent ús d'una mostra de nombres aleatòris, quants decimals correctes s'obtenen? Preneu mostres de tamany
clearvars; clf;
format short
a = 0; b = 1; p = 8;
g = @(x)(2./(1+x.^2));
for i=1:6
M = p^i;
x = rand(M,1);
y = g(x);
I_m(i) = (b-a)/M*sum(y);
end
dc = abs(I_m-pi/2);
disp(array2table([1:i;p.^(1:i);I_m;dc]',"VariableNames",{'i','M','I_aprox','error absolut'}))
